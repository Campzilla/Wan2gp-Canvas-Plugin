# Wan2GP Canvas Plugin 🎨

A powerful and easy-to-use Canvas Editor plugin for Wan2GP. This plugin provides an interactive drawing board right inside the Wan2GP interface, allowing you to create, tweak, and send images directly to the Generator with no hassle.

## ✨ Features

- **Interactive Drawing Tool**: A fully featured canvas to sketch and draw ideas directly within the Wan2GP UI. 
- **Expanded Brush & Eraser**: Increased size limits up to 200px for comfortable high-resolution editing.
- **Easy Image Editing**: Import existing pictures, modify them, and add new elements on the fly. Paste directly from clipboard or clear the layer with a single click via new Image Layer Controls.
- **Image Transformations**: Easily Fit Image, Center, Reset, H-Flip / V-Flip, and Rotate 90° for quick adjustments under the View Canvas controls.
- **Filled Shapes**: Draw solid rectangles and circles using the new "Fill Shape" property toggle.
- **Merge/Bake Sketched Layers**: Click to fuse your brush strokes and shapes directly onto the background photo, combining them into a single, durable layer.
- **Seamless Integrations**:
  - **Send to Reference Images**: Send your final creation straight to Wan2GP's "Inject Reference Images > Conditional Images (People / Objects)" with a single click. No need to save, download, or re-upload manually!
  - **Send to Inpainting**: Automatically falls back to standard Generator image input if Reference Image components are not available.
- **Plug & Play**: Simple and clean professional interface that runs smoothly without complex setups.

## 📦 Installation

Open Plugin Manager in Wan2GP and paste:

https://github.com/Campzilla/Wan2gp-Canvas-Plugin

or 

1. Download this repository (click on the green **Code** button -> **Download ZIP**).
2. Extract the downloaded `.zip` file.
3. Move the extracted folder inside the `plugins/` directory of your Wan2GP installation.
4. Restart Wan2GP and you're good to go!

You will find a new **🎨 Canvas Editor** tab ready to use directly in your interface.
